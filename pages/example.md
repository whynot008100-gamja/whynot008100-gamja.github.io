# 첫 번째 게시글

이것은 GitHub Pages 정적 블로그의 샘플 게시글입니다.

## 소개

이 블로그는 다음과 같은 기술로 구축되었습니다:

- **HTML, CSS, Vanilla JavaScript**: 프레임워크 없이 순수 웹 기술 사용
- **marked.js**: 마크다운 파싱
- **Prism.js**: 코드 하이라이팅
- **Giscus**: GitHub Discussions 기반 댓글 시스템

## 기능

### 1. 마크다운 지원

이 블로그는 마크다운 문법을 완전히 지원합니다:

- **볼드 텍스트**
- *이탤릭 텍스트*
- [링크](https://example.com)
- `인라인 코드`

### 2. 코드 하이라이팅

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}

greet("World");
```

```python
def hello_world():
    print("Hello, World!")

hello_world()
```

### 3. 목록

순서 없는 목록:
- 항목 1
- 항목 2
- 항목 3

순서 있는 목록:
1. 첫 번째
2. 두 번째
3. 세 번째

### 4. 인용문

> 이것은 인용문입니다.
> 여러 줄에 걸쳐 작성할 수 있습니다.

## 결론

이 블로그는 간단하고 효율적인 정적 블로그 솔루션입니다. 마크다운으로 게시글을 작성하고, GitHub에 푸시하면 자동으로 배포됩니다.
